# Görsel Eşleştirme Yardımcı Dosyası

## Kullanım

1. `show-images-for-mapping.html` dosyasını tarayıcıda aç
2. Her görselin altına ürün ismini yaz
3. "Eşleştirmeyi Oluştur" butonuna tıkla
4. Çıkan JSON'u kopyala ve bana gönder

## Alternatif: Manuel Eşleştirme

Aşağıdaki formatta bana gönderebilirsin:

```
ID: 77733815 -> Ürün: Mercimek Çorbası
ID: 77733816 -> Ürün: Tavuk Suyu Çorbası
...
```

Ya da daha basit:

```
77733815: Mercimek Çorbası
77733816: Tavuk Suyu Çorbası
77733817: Kelle Paça Çorbası
...
```
